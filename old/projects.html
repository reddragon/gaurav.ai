I have created / jointly created / contributed to a few projects in the past and continue to maintain some. Here I list a few of the recent ones.

bloomfilter.go
A bloom-filter implementation in Golang, which offers a standard bloom filter, a counting bloom filter and a scalable bloom filter (wherein, the bloom-filter can continue taking new elements without hurting the false positive rate, in a transparent manner).
Packed Memory Array
An implementation of the Packed Memory Array, which is a a datastructure that allows keeping elements in sorted order with an amortized insertion complexity of O(log2n), and requires only logarithmic extra space.
elixir
This was a fun project to write a simple Golang-based fortune cookie server.
Grid Computing Framework
An easy-to-use and deploy grid computing middleware.
Four Seasons of Code
A student project management application.
Genome Compression
I worked with Aniruddha and Madhava in this project to improve the compression of human genome sequence, by building upon the DNAZip project. Over the course of the project, we improved the compression by about 11%. Report (pdf).
Online Grading System
A programming contest grader originally written by Arijit De, modified by me for use in the TSEC Coding League with the 'Leagues' format.
